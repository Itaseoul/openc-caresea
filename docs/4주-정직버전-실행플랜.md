# 4주 실행 플랜 — 정직 버전 (시제품 + 비전 + 데이터 데모)

작성 2026.06.30. 외부 제안의 "4주 완성 플랜"을 이사회가 RESHAPE한 버전이다. 핵심 정정 하나 — **"3주차 소하천 실전 테스트"는 점용·안전·주파수 규제상 4주 내 합법적으로 불가능**하다. 4주에 진짜 손에 쥐는 것은 완성된 하천 로봇이 아니라 "통제 환경 시제품 + 정지형 붐 목업 + 정직한 데이터 대시보드 + 비전"이다. 이 정직함이 오히려 기술 신뢰를 만든다(`physical-ai-0stage-impl`의 라벨0 정직 프레임).

## 1주차 — 오픈소스 소싱

가장 빠른 오픈소스 HW/AI를 확보한다. 처음부터 설계하지 않는다. **코어는 이동 보트가 아니라 정지형 붐 + 고정 컨베이어**다(이사회 결정: 붐=데이터공장, 이동에 가치 없음). AIHub 하천 쓰레기 데이터셋과 학습 완료된 YOLOv10/11-nano를 확보하되 도메인갭(흙탕물·국내 성상)을 정직히 표기한다. 구매: Raspberry Pi 5 8GB + AI Kit(Hailo-8L) ≈ 20~24만, 카메라, 로드셀, 필라멘트(1기 BOM `소하천-수거로봇-1기-BOM.md` 레벨 A 약 30만 기준).

## 2주차 — 팩토리 1호점 + 에지 AI

이타 사무실 한쪽을 마이크로 팩토리 1호점으로 지정, 3D 프린터로 붐 목업·수거함 부품 출력, 제작 타임랩스 촬영. 에지 AI 테스트: Pi5+Hailo에 모델을 올려(.hef) **흙탕물 포함** 조건에서 편의점 컵·막걸리 병·비닐을 실시간 인식하는지 확인(맑은 물만으로는 약하다).

## 3주차 — 통제 환경 데모 + 정직 대시보드

하천 실전 대신 **사무실 수조/통제 환경**에서 붐·컨베이어 목업 위로 떠내려오는 플라스틱을 카메라가 인식하는 순간을 영상 확보. 대시보드 프로토타입을 연결하되 화면에 **"데모 데이터 / 학습모델 아님(라벨0)"** 정직 라벨을 박는다. 가능하면 drifter 1회 표류실측(`GPS-드리프터-표류실측-심층스터디.md`)으로 예측 정답라벨 첫 점을 만든다.

## 4주차 — 팩토리 오픈데이 + 부품 입양 펀딩

오픈데이: 환경·IT 인플루언서·대학생 개발자·잠재 후원 기업을 사무실(공장)로 초청, 3D 프린터 가동·에지 칩 분별을 라이브로. 부품 입양 펀딩 개시 — **BOM 보정 단가**로(`유튜브-톤가이드-슈퍼챗-설계.md` 환산표): "10만 원 = AI 추론칩 1개, 100만 원 = 자립 수거 스테이션 1대 입양." 지하철 게임 MVP(`지하철게임-MVP-스펙.md`)도 동시 공개해 QR 바이럴.

## 1달 산출물 (정직)

동작 시제품 1대(통제 환경 인식, 완성 하천 로봇 아님). 팩토리 스토리텔링 소스(영상·공간 사진). 정직 데이터 대시보드 프로토타입(라벨0 표기). 부품 입양 펀딩 페이지 + 게임 MVP. **완성 로봇을 약속하지 않고 "프로토타입 + 마이크로 팩토리 비전"을 증명**하는 것이 4주의 목표다.

## 병행 트랙 (규제·파트너, `첫대상하천-선정기준표.md`)

4주 데모와 동시에 부산 2트랙(괴정천 붐 / 감전천 데이터) 규제·부지 트랙을 착수한다 — `/permit` 문서팩, 데이터 MOU, ⑫ 공익법인 자문. 실제 하천 투입은 이 트랙이 열린 뒤(수개월)이며, 4주에 욱여넣지 않는다.

## 정직 경계

"Fake-it 대시보드"는 데모임을 명기한다. 4주 산출물을 "완성 로봇"으로 홍보하지 않는다 — 과장은 기술 신뢰를 깎는다. 통제 환경 인식 성공이 곧 하천 자율 수거를 뜻하지 않음을 분명히 한다.
