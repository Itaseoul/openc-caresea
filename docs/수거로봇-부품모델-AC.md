# 수거 로봇 부품 모델 예시 — 레벨 A(측정)·C(자립 수거)

작성 2026.06.30. BOM([[opensource-collection-robot]], `docs/소하천-수거로봇-1기-BOM.md`)의 레벨 A(약 30만)와 C(약 100만)를 **실제 제품 모델**로 내린다. 모델은 예시이며 치환 가능하고, 가격은 추정이다(국내 메이커 유통 디바이스마트·엘레파츠·메카솔루션 기준, 환율 약 1,350원/$). 검증 단가 앵커(Pi5·Hailo)는 스터디 5장.

## 인터페이스·펌웨어 메모

- 로드셀 → HX711 → Pi GPIO(클럭/데이터 2핀). 초음파 JSN-SR04T → GPIO(트리거/에코) 또는 UART 변형. 모터 → BTS7960 → Pi PWM. 카메라 → CSI(또는 USB).
- 인지 = YOLOv11n을 Hailo(.hef)로 변환해 AI HAT+에서 추론([[physical-ai-0stage-impl]]의 perception `det-v0`). 출력(적재율·성상·잼)은 관측 레코드(`observation.ts`)로 적재 → 수거 중량 앵커로 보정.

## 안전(결정형, AI·전자 독립)

- 컨베이어는 적재 임계 초과에서만 구동하고 잼·고전력 감지 시 정지·역회전한다(에너지·마모 절약).
- 홍수 통수 fail-open(붐 한쪽 기계식 분리)은 AI·전자와 독립으로 작동한다. 데이터는 임계값 보정에만 쓴다(스터디 6장, 전문가 패널).

## 다음

1. 컨베이어 폭·인양 높이를 괴정천 단면 기준으로 확정 → 벨트·모터·프레임 규격 고정.
2. 모터 듀티 사이클 산정 → 태양광·배터리 용량 확정(현재 100W·30Ah는 보수적 여유 가정).
3. 레벨 A부터 조달해 0단계 측정·perception을 먼저 가동(감전천), C는 괴정천에서.
