# 지하철 게임 MVP 스펙 — "마이크로 팩토리 타이쿤"

작성 2026.06.30. 이타 자문 이사회 검토(`spectacle-quarantine-review`)에서 KEEP 판정된 "지하철 캐주얼 게임" 제안의 실행 스펙이다. 단 하나의 안전 원칙이 설계를 지배한다 — **게임은 실제 로봇을 움직이지 않는다.** 게임은 디지털 마중물이고, 실제 구동계와 분리되어 있어야 안전·법적 리스크가 0이 된다.

## 0. 콘셉트

출퇴근 자투리 시간(10~30분), 한 손 세로 조작의 캐주얼 방치형/타이쿤. 유저는 가상의 '이타서울 마이크로 팩토리' 공장장이 되어 3D 프린터로 부품을 뽑고 추론칩을 업그레이드해 가상 소하천에 수거 스테이션을 파견한다. 플레이하며 physical AI 지식을 자연스럽게 학습한다("Hailo 칩으로 바꿨더니 인식률↑", "정지형 붐이 이동 보트보다 에너지·안전 우위").

## 1. 핵심 루프

방치형 경영: 부품 제작 → 스테이션 배치 → 수거량 누적 → 추론칩·태양광·컨베이어 업그레이드. 1기 BOM(`소하천-수거로봇-1기-BOM.md`)의 실제 부품·레벨(A 측정노드 / C 자립수거 / D 정식)을 게임 업그레이드 트리로 그대로 매핑한다 — 학습과 게임이 일치한다.

## 2. 실제 임팩트 데이터 피드 (이타 특화)

실제 SEA:CUT 수거량이 게임에 보너스로 들어온다. "실제 괴정천 붐이 PET 100개를 차단 → 전 유저 클린 포인트 지급" 푸시. 가상 게임과 실제 환경 정화가 연결되는 효능감이 핵심 차별점이다. 데이터 소스는 `/api/observations`·`litter-risk`이고, **읽기 전용 피드**다(게임→현장 제어는 없다).

## 3. 안전·분리 원칙 (협상 불가)

게임의 어떤 입력도 실제 로봇·붐을 구동하지 않는다. 슈퍼챗 "부스터" 같은 유료 실시간 조종은 게임에도 이식하지 않는다(제안 D KILL과 동일). 게임은 *관전·학습·기부 유도*까지다.

## 4. 수익→기부 모델

보상형 광고(제작 시간 단축용) 수익을 실제 부품비로 적립. ⚠️ 광고 수익 기부 비율·세무(지정기부금)·플랫폼 정책은 ⑫ 공익법인 자문(`공익법인-자문-질문리스트.md` A)에서 확정 후 표기. 매칭 챌린지("게임 내 정화율 100% 달성 시 후원 기업이 실제 추론칩 10대 기부")로 기업 CSR 연결.

## 5. 기술 스택 (빠른 출시)

웹 우선(HTML5 / Phaser) — 앱 다운로드 없이 카톡 링크·이타 웹에서 바로 실행, 접근성 극대화. 2D 플랫/도트로 아트 비용 최소화. 이타 기존 웹 플랫폼의 한 메뉴로 임베드 가능.

## 6. 4주 MVP (로봇·팩토리 플랜과 병행, `4주-정직버전-실행플랜.md`)

1~2주: "쓰레기 받기" 초단순 미니게임(YOLO 조준 시각효과). 3주: 종료 화면에 "이 게임 속 로봇을 실제로 만드는 중" + 통제환경 데모 영상 링크. 4주: 팩토리 오픈데이 QR 플레이 + 점수 랭킹전으로 바이럴.

## 7. 정직 경계

게임 수치(가상 정화율·포인트)와 실제 임팩트(실측 수거 kg)를 화면에서 분리 표기한다 — 혼동 금지. 광고 수익의 기부 비율을 명시한다. 게임은 실제 로봇을 조종하지 않음을 UI에 명기한다. 보상은 굿즈·기부 인증서·환경 포인트로 하되 실물 약속은 재고·배송 가능 범위로 한정한다.
