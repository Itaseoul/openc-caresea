# 첫 대상 하천 선정 기준표

작성 2026.06.30. 커뮤니티 동호회(`커뮤니티-동호회-연계-제안.md`)와 유튜브 라이브(`유튜브-톤가이드-슈퍼챗-설계.md`)가 공유하는 "첫 대상 하천 1곳" 결정을 위한 의사결정 표다. 핵심 긴장은 **참여 도달(서울 접근성·인지도) vs 기투자 인프라(부산 붐·permit·데이터)**다. 정직하게 말하면 둘은 같은 곳이 아니다.

## 평가 축

기존 자산 연계(붐·`/permit`·CCTV·드리프터·HRFCO 기투자)가 가장 무겁다. 그다음 안전창 운영 가능성(실시간 수위 데이터 존재 여부 — 안전 게이트의 전제), 규제 경량(소하천 > 국가유산·습지), 이타 운영 접근성(정기 출동 빈도), 도달·상징성(유튜브·시민 인지), 동호회 생태계 존재(카약·낚시 카페) 순이다.

## 전문가 결정 (확정)

**부산 2트랙 집중으로 확정한다.** 물리 코어는 괴정천(붐 수거 레벨 C)·감전천(데이터 노드 레벨 A)이고, **참여·유튜브·동호회 1차 앵커도 부산**(괴정천 현장 + 감전천 + 낙동강 하구 footing)에 둔다. 입지를 쪼개지 않는다.

근거. 이 프로젝트는 ChangeX 부산 수행 증빙·부산시 해양정책과 채널·5/31 바다의 날 정화로 **이미 부산 운영 footing**이 있어, "서울 조직이라 부산 출동 빈도가 부족할 것"이라는 우려가 작다. 따라서 이전 검토의 "서울 안양천 참여 앵커(분리 운영)" 안은 철회한다 — 코어가 증명되기 전 분산은 비용만 키운다. `/permit`·HRFCO·부산시 채널이 이미 부산에 깔려 있고, 을숙도는 4중 규제라 코어가 아니다.

서울 확산은 후순위다. 부산 코어가 안전창 수거·예측 검증을 1회라도 성공시킨 뒤, 그 영상을 들고 안양천을 "전국 확산 1호 거점"으로 연다.
