## 0. 한 줄 요약

이 부클릿은 **10–16세 청소년이 과학 훈련 없이도 비교가능한 해변 플라스틱 데이터를 생산**하도록 설계된 완성형 시민과학 프로토콜이다. 핵심 3장치 — (1) **모든 지점 동일 방법 + 무작위 표본**으로 비교가능성 확보, (2) **강제 사진**을 표본 채택의 전제조건으로 삼는 품질 게이트, (3) **연구 파트너의 사후 과학 평가·발행**. SEA:CUT이 소하천·하구 정화 참여자에게 이식하려는 "표준 관측 카드 + 사진 검증 게이트"가 이미 여기에 구현되어 있다.

## 1. 개요·구조

| 항목 | 내용 |
|---|---|
| 대상 | 청소년 10–16세(강 부클릿은 10–18세) |
| 그룹 규모 | 6~30명(학급·동아리·클럽) |
| 소요 | 준비·후속 포함 약 3일 / 6~8교시 + 표본채취 약 2시간 |
| 기원 | 킬 사이언스 팩토리(독일) 2016, BMBF "바다의 해 2016·17" → 2018 "환경 속 플라스틱" → 2020 트리오 의장국(독·포르투갈·슬로베니아) → 2022.1 EU 집행위 지원으로 13개국 확대(기사·후속 집계는 14개국) |
| 발행·집필 | DLR 프로젝트 관리기관(본). 편집 파트너 6개국 기관(아래 §8) |
| 문의 | plastic-pirates@dlr.de · #PlasticPiratesEU |

**과학의 5단계**(부클릿이 청소년에게 직접 가르치는 틀):
1. 연구 질문 세우기 → 2. 가설 세우기 → 3. 연구 방법 계획 → 4. 실험·데이터 수집 → 5. 결과 평가·비교.

**비교가능성 원칙(핵심)**: "표본을 유럽 전역 다수 청소년이 여러 나라에서 채취하므로 **모두가 정확히 같은 방법**을 지켜야 한다. … 모든 채취 지점에서 같은 방법을 써야 나중에 데이터를 **비교**할 수 있다." + 무작위 표본(쓰레기가 많거나 적은 곳을 고르지 말 것)으로 대표성 확보.

## 2. 다섯 그룹 분업 (표본 채취 설계)

| 그룹 | 임무 | 권장 인원 | 핵심 방법 | 연구 질문(요지) |
|---|---|---|---|---|
| **A. 해변 위 쓰레기** | 정량 | 4~6 | 50m×20m 구역에 **트랜섹트**(해측→후면), 3개 해변대(A 조간대 / B 상조대 / C 식생대)마다 **3×3m 방형구**, 담배꽁초 크기(2~3cm) 이상만 수거·계수, m²당 환산 | 얼마나? 재질은? 바다로 유입 가능성은? |
| **B. 쓰레기 다양성** | 분류·정량 | 6~8 | **분류 스테이션** 설치, ~25개 카테고리로 분류·계수·**무게 측정**, 일회용 플라스틱 비율 산출 | 최다 카테고리? 일회용 비율? 어떤 (정책)조치가 효과적? |
| **C. 리터 포렌식** | 출처추적 | 3~4 | 문자 있는 품목 **촬영**, 언어·통화·브랜드로 **원산지·제조사 식별**, 열화도(저/중/고) 기록 | 지역 쓰레기 출처? 책임 브랜드? |
| **D. 리포터 팀** | 검증·메타 | 4~6 | 전 과정 **사진·영상 기록**, 쓰레기 발생원 식별, 최근 7일 기상, **좌표(십진도)**, 단체 사진, 기사 작성 | 표본이 성공적이었나? 발생원은? 퇴적 요인은? |
| **추가. 미세플라스틱** | 미세 | 4~6 | 만조선 따라 1×1m 방형구, 5cm 굴착, **체(망목 1mm)** 로 미세플라스틱 분리·계수, m²당 환산 | 얼마나? 어떤 카테고리? |

> 그룹 D "리포터=검증"은 Horizon Magazine 기사가 강조한 "4번째 그룹=검증"과 정확히 일치. 다만 부클릿을 보면 **검증은 D 한 그룹만의 일이 아니라, 모든 그룹이 사진을 남기고 D가 총괄·대조**하는 구조다.

## 3. ⭐ 사진 게이트 — SEA:CUT 포지셔닝의 결정적 근거

부클릿은 사진을 단순 기록이 아니라 **표본이 과학 연구에 채택되기 위한 전제조건**으로 규정한다. 즉 프로토콜 레벨의 "게이트"가 이미 존재한다:

- **그룹 A**: "쓰레기가 없어도 각 표본점을 반드시 촬영하라(빈 천의 종이 사진). **그러지 않으면 여러분의 결과는 과학 연구에 포함될 수 없다**." → 사진 = 채택 전제조건.
- **그룹 A**: 종이(트랜섹트 번호·표본점·학교명)와 천 위에 펼친 쓰레기를 함께 촬영, 조각이 겹치지 않고 식별 가능해야 함.
- **그룹 B**: 카테고리 이름과 함께 촬영 후 **사진 품질 확인**, 양 많으면 여러 장.
- **그룹 C**: 문자 판독 가능하게 촬영해야 원산지 식별 가능.
- **업로드**: 핵심 데이터 + **사진**을 업로드, 결과지(20·21쪽) 스캔/사진도 업로드, 캠페인 종료 후 **2주 내** 제출.

**SEA:CUT 정정 결론**: 앞서 "사진 검증 게이트는 우리 프레이밍, 강제 로직 미확인"이라 유보했으나 — **부클릿이 명시적으로 사진을 표본 채택의 필수 게이트로 규정**한다. 따라서 "EU 프로토콜이 이미 채택한 강제 사진 게이트를, SEA:CUT은 업로드 시 자동 EXIF/geo 검사로 **자동화·엄격화**한다"가 정확하고 강력한 프레이밍이다.

## 4. 폐기물 카테고리 체계 (그룹 B) — 우리 관측 스키마로 직접 채용

일회용 플라스틱 비율 산출을 위해 아래 카테고리로 분류·계수한다. **SEA:CUT `/api/observations` 품목 카테고리로 그대로 채택 가능**(비교가능성·국제 정합).

- **플라스틱(일회용 집계 대상)**: 비닐봉투 / 음료 페트병 / 페트병 뚜껑 / 테이크아웃·패스트푸드 포장(일회용 커피컵·뚜껑 포함) / 플라스틱 수저·접시(젓개·빨대 포함) / 과자·비스킷·칩 포장 / 면봉(Q-tip) / 물티슈·탐폰·생리대 / 스티로폼(폴리스티렌)
- 소형 플라스틱 <2.5cm / 식별불가 기타 플라스틱
- **금속**: 음료캔 / 병뚜껑 / 알루미늄 포일 / 기타 금속
- **유리**: 음료 유리병 / 유리 조각 / 기타 유리
- **기타**: 담배꽁초 / 종이 / 섬유(의류·신발) / 고무(타이어·고무밴드) / 풍선 / 식별불가 기타 / **지역 쓰레기(Local waste)**

**계수 규칙**: 느슨히 연결·용기 안 품목은 개별 계수(봉투 안 내용물 각각), 단단히 결합·밀착 포장(뚜껑 잠긴 유리병·감긴 그물)은 1개로, 더 큰 물체가 카테고리 결정. 젖은 플라스틱은 말려서 계량.

## 5. 표본 메타데이터 = 표준 관측 카드 필드

**업로드 필드**(플랫폼 data-upload 폼): 그룹명 · 단체 사진(동의 전제) · 참여자 수 · **표본 채취일** · **강/하천 이름** · **표본 위치**. → SEA:CUT 1페이지 관측 카드 필드와 1:1 대응.

**해변 프로파일(그룹 D 메타)**: 해변 유형(도시/준도시/자연) · 후면 지형(절벽/사구/암석/숲/농경지/도로 등) · 향(북동남서) · 해안선 곡률(직선/오목/볼록/파상) · 기저 조성(모래/자갈/암반 %) · 경사 · 접근성 · 최근 7일 기상 · **좌표(십진도)**. → 소하천 스팟 메타로 이식(발생원×트랩 예측기 `/api/litter-risk` 입력 보강).

**발생원 분류(그룹 D)**: 주민(넘치는 쓰레기통·생활폐기물) · 방문객(파티 쓰레기·바비큐·빈 병) · 무단투기 · 산업 · 농업(비닐·온실 피복) · 해운(선상 용품·방수복) · 어업(그물·낚싯줄·스티로폼 상자). → 우리 "발생원(source)" 라벨 체계로 채용.

## 6. 결과 비교·후속 활동

- **비교 축**: 내 결과 vs 자국 평균 vs 유럽 평균 — m²당 쓰레기 수, 일회용 플라스틱 %, 최다 브랜드, 미세플라스틱/m². (플랫폼 지도에서 자동 대조)
- **오염 자가평가**: 1~6 스케일.
- **후속(행동 전환)**: Reduce(감량) / Plastic-free(포장 없는 소비) / Upcycling(업사이클) / Outreach(정책결정자 인식 제고). → 교육·시빅 임팩트를 데이터 생산과 결합.

## 7. SEA:CUT 이식 요약 (실행 항목)

1. **표준 관측 카드 1페이지** 제작 — 부클릿 업로드 필드(강명·위치·날짜·참여자·사진) + 발생원 분류 + 좌표(십진도)를 그대로.
2. **품목 카테고리** = 부클릿 그룹 B의 ~25종을 `/api/observations` 스키마로 채택(국제 비교가능성 확보).
3. **사진 게이트 자동화** — 부클릿의 "사진 없으면 표본 제외"를 업로드 시 자동 EXIF/geo/timestamp 검사로 구현. 통과분만 라벨0→라벨1 승격([[physical-ai-0stage-impl]]).
4. **비교가능성 규칙** — "모든 지점 동일 방법 + 무작위 표본" 문구를 우리 프로토콜에 명시.
5. **오픈데이터 발행** — 라벨1 데이터를 DOI 저장소로(Zenodo 선례) → 공공기관 인용 권위([[public-agency-focus]]).
6. **정책 훅** — 부클릿이 이미 "어떤 정책이 일회용을 줄이나"를 연구 질문으로 내장. 우리 리포트에 동일 질문 축 채용.
